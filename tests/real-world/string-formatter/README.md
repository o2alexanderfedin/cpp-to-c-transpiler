# String Formatter - Real-World Test Project

## Overview
Type-safe string formatting utility demonstrating:
- Templates and variadic templates
- Operator overloading
- Move semantics
- Template specialization
- Type traits

## Features
- Printf-style formatting with type safety
- Stream-style formatting (<<)
- Custom format specifiers
- Support for custom types via specialization
- Compile-time format string parsing (C++17)

## C++ Features Tested
1. **Variadic Templates**: Arbitrary number of arguments
2. **Template Specialization**: Custom formatting for types
3. **Operator Overloading**: << operator for chaining
4. **Move Semantics**: Efficient string building
5. **Type Traits**: Compile-time type inspection
6. **STL**: std::string, std::ostringstream, std::vector

## Target Size
2,500-4,000 LOC (with extensive examples and type support)
