TEST EXECUTION FLOW
================================================================================

Each test follows this pipeline:

Stage 1: PARSE
  ├── Input: C++ source code with operator overloads
  ├── Tool: Clang frontend
  └── Output: C++ AST

Stage 2: TRANSLATE
  ├── Input: C++ AST
  ├── Tools: FunctionHandler, ExpressionHandler, StatementHandler
  │         ComparisonOperatorTranslator, NameMangler
  └── Output: C AST (operator calls → function calls)

Stage 3: EMIT
  ├── Input: C AST
  ├── Tool: CodeGenerator
  └── Output: C source code

Stage 4: COMPILE
  ├── Input: Generated C code
  ├── Tool: gcc -std=c99
  └── Output: Executable binary

Stage 5: EXECUTE
  ├── Input: Compiled binary
  ├── Method: ./executable
  ├── Capture: Exit code
  └── Output: Test PASS/FAIL based on exit code

Stage 6: CLEANUP
  ├── Delete temporary C++ file
  ├── Delete temporary C file
  └── Delete executable

   add_executable(ComparisonOperatorsIntegrationTest
       comparison-operators/ComparisonOperatorsIntegrationTest.cpp
       ${CMAKE_SOURCE_DIR}/../../src/handlers/HandlerContext.cpp
       ${CMAKE_SOURCE_DIR}/../../src/handlers/FunctionHandler.cpp
       ... (other dependencies)
   )
   target_link_libraries(ComparisonOperatorsIntegrationTest GTest::gtest GTest::gtest_main ${CLANG_LIBS})
   target_include_directories(ComparisonOperatorsIntegrationTest PRIVATE ...)

   gtest_discover_tests(ComparisonOperatorsIntegrationTest
       WORKING_DIRECTORY ${CMAKE_CURRENT_SOURCE_DIR}
       PROPERTIES LABELS "integration;comparison_operators;phase51"
   )

2. Update status messages in CMakeLists.txt:

   message(STATUS "  - ComparisonOperatorsIntegrationTest: 15 tests (Phase 51)")
   message(STATUS "  Total: XXX integration tests")

3. Run tests:

   cd build
   ctest -R ComparisonOperators
   # or run specific test:
   ./ComparisonOperatorsIntegrationTest --gtest_filter="*BubbleSort*"
