Overview

This project implements a C++ to C transpiler that produces high-quality, human-readable C code suitable for formal verification with tools like Frama-C. The converter handles modern C++ features including classes, templates, RAII, exceptions, RTTI, lambdas, and C++20 coroutines.

Key Features

Two-Phase Translation

Intermediate C AST generation for optimal code quality and maintainability

STL Self-Bootstrapping

Automatic conversion of STL containers through template instantiation

PNaCl SJLJ Exceptions

Thread-safe exception handling with action tables

Formal Verification Ready

Clean output optimized for Frama-C verification
